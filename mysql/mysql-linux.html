Linux环境安装MySQL8.0

2019-03-25

Linux查看当前操作系统版本信息

  • 指令
    • cat /proc/version
  • 输出
    • Linux version 3.10.0-957.el7.x86_64 (mockbuild@kbuilder.bsys.centos.org) (gcc version 4.8.5 20150623 (Red Hat 4.8.5-36) (GCC) ) #1 SMP Thu Nov 8 23:39:32 UTC 2018

MySQL下载

  • 官网下载地址 https://dev.mysql.com/downloads/mysql/

  • 选择合适的版本进行下载,64位系统则下载mysql-8.0.15-linux-glibc2.12-x86_64.tar.xz

  • 选择只下载，不进行登录或注册

MySQL安装

解压mysql-8.0.15-linux-glibc2.12-x86_64.tar.xz文件，因为是xz的压缩格式，所以需要先进行xz -d解压，然后才进行 tar -xvf 操作

  • 解压xz文件

    • xz -d mysql-8.0.15-linux-glibc2.12-x86_64.tar.xz

  • 解压tar文件

    • tar -xvf mysql-8.0.15-linux-glibc2.12-x86_64.tar

  • 将解压出的文件夹改名为mysql

    • mv mysql-8.0.15-linux-glibc2.12-x86_64 mysql

  • 添加系统mysql组和mysql用户

    • groupadd mysql
    • useradd -r -g mysql mysql

    useradd 指令解释

    －g：指定用户所属的起始群组。
    －r ：建立系统账号。

  • 改变 mysql 目录权限

    • chown -R mysql:mysql ./

    chown 指令解释

    chown [选项]… [用户名] ：[用户组]] 文件…

  • 初始化数据库

    • mysql/bin/mysqld –initialize –user=mysql –basedir=/home/mysql/mysql –datadir=/home/mysql/mysql/data

    执行mysqld –initialize命令进行安装，值得注意的一点是这里和低版本mysql安装命令不同；basedir和datadir选择自己文件夹的结构。

    另外这里会生成随机密码，一定要记录下来。

    2019-03-20T16:08:30.923330Z 5 [Note] [MY-010454] [Server] A temporary password is generated for root@localhost: HJHr>lkQS2fg

    最后的随机字符串就是随机密码。

  • 修改my.cnf

    • vi /etc/my.cnf

    • basedir = /home/mysql/mysql

      datadir = /home/mysql/mysql/data

      port = 3306

      socket = /tmp/mysql.sock

      character_set_server = utf8mb4

  • 建立MySQL服务

    • cp -a ./mysql/support-files/mysql.server /etc/init.d/mysqld
    • chmod +x /etc/rc.d/init.d/mysqld
    • chkconfig –add mysqld

    可以使用chkconfig –list 检查服务是否生效

  • 配置环境变量

    • vi /etc/profile

      在 profile 文件底部添加如下两行配置

    • PATH=/home/mysql/mysql/bin:/home/mysql/mysql/lib:$PATH

    • export PATH

    设置环境变量立即生效

    • source /etc/profile

  • 启动MySQL服务

    • service mysqld start

  • 登录并修改密码 (MySQL8的密码认证插件默认是caching_sha2_password,所以需要特别指定mysql_native_password插件，见下条)

    • mysql -uroot -p
    • 输入初始化数据库时随机生成的密码，如果没记住也没关系，删除data目录下的文件然后重新初始化一次
    • alter user ‘root‘@’localhost’ identified by ‘jin’;

  • 开启mysql远程访问权限

    • use mysql;
    • select host, user from user; 查看信息
    • update user set host = ‘%’ where user = ‘root’;
    • ALTER USER ‘root‘@’%’ IDENTIFIED WITH mysql_native_password BY ‘jin’;
    • FLUSH PRIVILEGES;

  • 完成安装
